# 🌟 Creator Economy on Stellar (Soroban)

## 📌 Project Description

This project is a simple **Creator Economy smart contract** built using **Soroban** on the Stellar network. It enables creators to register on-chain and receive support from fans in the form of tips, creating a decentralized and transparent monetization system.

The goal is to empower creators by removing intermediaries and giving them direct access to their audience through blockchain technology.

---

## 🚀 What it does

* Allows users to register themselves as creators
* Enables fans/supporters to send tips to creators
* Tracks total earnings of each creator on-chain
* Stores creator data securely using Soroban smart contract storage

This contract demonstrates the foundational building blocks of a decentralized creator platform.

---

## ✨ Features

* 🔐 **Creator Registration**
  Creators can register their wallet address on-chain

* 💸 **Tipping System**
  Supporters can tip creators directly

* 📊 **Earnings Tracking**
  Transparent tracking of total earnings per creator

* ⚡ **Lightweight & Gas Efficient**
  Built with Soroban’s optimized smart contract environment

* 🧩 **Extensible Design**
  Can be expanded with subscriptions, NFTs, memberships, etc.
